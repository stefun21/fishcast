# FishCast 3.1 — Catalog Pro

- pagină separată `/lakes` cu toate locațiile;
- geolocație solicitată automat și sortare implicită după distanță;
- API intern paginat pentru cataloage mari;
- filtre după tip, regim și nivelul de încredere;
- paginare responsive cu 12, 24 sau 48 de rezultate;
- întregul card al unei locații este accesibil prin click/tap;
- temă vizuală nouă, inspirată de apă și navigație;
- animații discrete și suport `prefers-reduced-motion`;
- navigație desktop și mobilă actualizată;
- ruta detaliilor este randată la cerere, pentru a evita build-uri uriașe;
- workflow stabil fără conexiuni către data.gov.ro.
