# FishCast România

FishCast este o aplicație PWA pentru descoperirea locurilor de pescuit din România, cu geolocație, hartă interactivă, vreme live și Fishing Index.

## Puzzle 14–15

Această versiune adaugă filtre avansate, rază configurabilă între 25 și 120 km, filtrare după tipul locației și calitatea datelor, cache local pentru ultimele rezultate și un API live mai rezistent, cu trei endpointuri Overpass, timeout, cache server și fallback la rezultate vechi.

## Deploy

Încarcă toate puzzle-urile în ordine în același repository GitHub. Vercel detectează automat proiectul Next.js. Nu sunt necesare variabile de mediu.
